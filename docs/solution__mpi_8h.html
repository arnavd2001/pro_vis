Procedure that the slave nodes should execute.

Consists of waiting for migrchs, calculating its fitness, and sending the fitness back to node 0. The fitness is sent back with the same MPI_TAG that was received with the migrch. The slave will return once the first element of the migrch received is equal 0xFF.
